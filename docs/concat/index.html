Les exercices de ce chapitre portent sur la précédence des opérateurs, ils démontrent, entre autres, les aspects suivants.

  • L'ordre de réalisation des opérateurs.
  • La possibilité de changer cette ordre en regroupant les opérateur entre parenthèses.
  • L'opérateur de concaténation de chaînes caractères.
